# Famigo Frontend（ファミゴー / フロントエンド）

Famigo（ファミゴー）のフロントエンド（SPA）リポジトリです。  

- デプロイURL： https://famigo-odekake.com
- バックエンドリポジトリ： https://github.com/tomo-taka108/famigo-backend


## AI利用について

このフロントエンドは、AI（ChatGPTをメインに、Gemini、Claudeも使用）を活用して実装したコードが中心です。  
そのうえで、要件への調整・UI微調整・挙動確認・デプロイ設定などは自ら行っています。


## 使用技術

- React **19.2.0**
- TypeScript **5.9.3**
- Vite **7.2.4**
- Tailwind CSS **4.1.18**（`@tailwindcss/vite` 利用）
- React Router DOM **7.10.1**
- lucide-react（アイコン）
